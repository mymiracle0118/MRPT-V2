## MarptoV2

https://book.getfoundry.sh/

## Setup

```shell
$ forge install
$ npm install
```

### Build

```shell
$ forge build
```

### Test

```shell
$ forge test
$ npm run test
```

### Gas Snapshots

```shell
$ forge snapshot
```

### Deploy

```shell
$ npm run deploy
$ npm run deployToSepolia
```
